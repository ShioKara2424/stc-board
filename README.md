# 🐱 Save the Cat Board

ブレイク・スナイダーの「セイブ・ザ・キャット」メソッドに基づく脚本構成ボードアプリ。

4つのACTレーンにカードとグループを配置して、物語の構成を視覚化します。

## 機能

- ACT 1〜3の4レーンにカードを自由配置
- グループ（点線枠）でカードをまとめて1シーンとしてカウント
- カラーパレット（10色）とタグ（8色、複数選択可）
- ゴーストカード（点線枠・シーンカウント除外）
- ライト／ダークテーマ切り替え
- Undo / Redo（最大80回）
- JSON形式で保存・読込
- PWA対応（オフライン利用可）

## 使い方

https://shiokara2424.github.io/stc-board/ にアクセス、またはローカルで `index.html` を開く。

## 技術

バニラJS、フレームワーク不使用、単一HTMLファイル。
